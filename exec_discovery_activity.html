PulseApp — web application tech stack & project description

PulseApp is an internal analytics dashboard that aggregates real-time operational metrics for business unit leaders. It is built on a three-layer architecture.

Architecture overview
Presentation

React 18 + TypeScript

Single-page application hosted on Vercel. Uses Recharts for data visualisation and Tailwind CSS for styling. Communicates exclusively via REST calls to the middle layer.

Middle layer

Node.js / Express API

Stateless REST API. Handles authentication (JWT), request validation, business logic, and data aggregation before passing results to the frontend. Deployed on AWS ECS.

Backend

PostgreSQL 15 + Redis 7

PostgreSQL stores transactional and user data. Redis is used as a caching layer for metric queries with a 5-minute TTL to reduce database load.

Key design decisions

Authentication model: JWT tokens issued by the middle layer with a 1-hour expiry. Refresh tokens stored in Redis.

Data access: All database queries go through the API — the frontend never talks directly to PostgreSQL.

Caching strategy: Metric aggregation queries are expensive. Results cached per user in Redis keyed by metrics:{userId}:{date}.

# PulseApp API

Three files cover the core server logic:

metricsRoutes.js — all metric endpoints. Owner: Priya S.

authMiddleware.js — JWT validation on every request.

Dashboard.tsx — primary React view, calls /api/metrics.

Run locally: npm install && npm run dev

// metricsRoutes.js — middle layer const router = require('express').Router(); const cache = require('./redisCache'); router.get('/api/metrics', auth, async (req, res) => { const key = `metrics:${req.user.id}:today`; const cached = await cache.get(key); if (cached) return res.json(cached); const data = await db.query('SELECT ...'); cache.set(key, data, 'EX', 300); res.json(data); }); router.post('/api/metrics/export', auth, ...); router.delete('/api/metrics/cache', adminOnly, ...); module.exports = router;
// authMiddleware.js const jwt = require('jsonwebtoken'); function auth(req, res, next) { const token = req.headers['authorization'] ?.split(' ')[1]; if (!token) return res.status(401).json({ error: 'Unauthorized' }); try { req.user = jwt.verify(token, process.env.JWT_SECRET); next(); } catch { res.status(403).json({ error: 'Invalid token' }); } } module.exports = { auth };
// Dashboard.tsx — presentation layer import { useEffect, useState } from 'react'; export default function Dashboard() { const [metrics, setMetrics] = useState(null); useEffect(() => { fetch('/api/metrics', { headers: { Authorization: `Bearer ${token}` } }) .then(r => r.json()) .then(setMetrics); }, []); return <MetricsGrid data={metrics} />; }
