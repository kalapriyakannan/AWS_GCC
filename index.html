Activity 01
Paradox of Distributed Knowledge
Navigate three separate artifacts — an engineering Wiki, a GitHub repository with live code, and a Google Doc — to answer two questions about a real project. No single artifact has the full picture. You have to piece it together.

Activity 02
Skills Compass
You have 10 AI skills, each doing one specific thing. Given a real-world job to be done, identify which skills — in which order — complete the workflow. Two scenarios of increasing complexity. Can you find the right sequence?
